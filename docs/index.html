Nokhal is not just another database. It is a privacy-first storage engine designed to run locally, giving developers full ownership of their data. No cloud, no tracking, no hidden layers. Everything is encrypted, fast, and under your control.

POWERED BY TITANIUM

AES-256-GCM

Military-grade encryption at rest. Every byte is sealed with authenticated encryption to ensure privacy and integrity.

Argon2id

Modern key derivation that resists GPU-based brute force attacks. Your passwords are the master key to your fortress.

Log-Structured

Append-only architecture designed for extreme write performance and crash consistency. Data is never lost.

Auto-Compaction

Intelligent background cleanup that reclaims space while maintaining database health without downtime.

main.go 
package main

import "github.com/wesleyyan-sb/nokhal"

func main() {
    // Create an encrypted fortress
    db, _ := nokhal.Open("vault.nok", "ultra-secret-pass")
    defer db.Close()

    // Store data in seconds
    db.Put("secrets", "api_key", []byte("sk_live_12345"))

    // Retrieve with speed
    val, _ := db.Get("secrets", "api_key")
    println(string(val))
}
