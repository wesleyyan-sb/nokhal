# Security Policy

## Supported Versions

We actively maintain and provide security updates for the latest stable versions of Nokhal.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

> Only the latest minor release is guaranteed to receive security updates.

## üîê Security Principles

Nokhal follows strict security practices:

- Uses **AES-256-GCM** for encryption  
- Relies only on **standard, well-tested cryptographic libraries**  
- Avoids custom or experimental cryptographic implementations  
- Ensures data integrity via checksums and safe write mechanisms
